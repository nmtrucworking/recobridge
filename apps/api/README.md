# RecoBridge API

FastAPI service implementing the contract in `docs/05_API/openapi.yaml`.

## Local development

```powershell
python -m pip install -e ".[test]"
$env:RECOBRIDGE_API_TOKEN="recobridge-demo-token"
uvicorn recobridge_api.app:app --reload --port 8000
pytest
```

Without `RECOBRIDGE_DATABASE_URL`, the service uses an isolated in-memory event
store for local development. For durable exposure and feedback ingestion, run
PostgreSQL 16 locally and set `RECOBRIDGE_DATABASE_URL` before starting the API.

## Load production model alias

The API accepts either a direct serving bundle or the checksum-protected
`production.json` alias generated by the ML promoter:

```powershell
$env:RECOBRIDGE_MODEL_BUNDLE_PATH="..\ml\artifacts\models\release\production.json"
uvicorn recobridge_api.app:app --port 8000
```

Benchmark the complete ASGI request path before promotion:

```powershell
recobridge-api-benchmark <serving-bundle> --output <performance-report>
```

The optimized release schema scores at most 200 precomputed candidates per
request and truthfully reports the baseline strategy when the XGBoost candidate
has not passed its promotion gates.
